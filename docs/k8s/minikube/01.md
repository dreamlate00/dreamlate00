pod-nginx.yml

```
apiVersion: v1
kind: Pod               # 创建的是Pod类型
metadata:               # 元数据, 一些基础信息
    name: nginx       # 名称
    labels:
        app: MyApp # 标签
spec:
    containers:
    -  name: pod-nginx     # 容器名称
       image: nginx:alpine # 使用的镜像
       ports:
       -  containerPort: 80 # 容器监听的端口号
```
